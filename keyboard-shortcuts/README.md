# SublimeText 2 Keyboard Shortcuts

### Application

Shortcut | Description |
| -------- | ----------- |
| <kbd>⌘</kbd> <kbd>,</kbd> | Preferences |

### Find/Replace

| Shortcut | Description |
| -------- | ----------- |
| <kbd>⌘</kbd> <kbd>F</kbd> | Find |
| <kbd>⌘</kbd> <kbd>⌥</kbd> <kbd>F</kbd> | Replace |
| <kbd>⌘</kbd> <kbd>⇧</kbd> <kbd>F</kbd> | Find in files |
